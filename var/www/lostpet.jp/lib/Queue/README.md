# queue
- 遅らせて実行したいプログラムを管理する。

## 一覧
| TYPE | 説明 |
| -- | -- |
| 1 | {empty} |
| 2 | {empty} |
| 3 | {empty} |
| 4 | 新着通知 (案件) |
| 5 | 削除通知 (案件) |
| 6 | 掲載期限を確認し、期限に達してたら削除する。 |
| 7 | debounce `CaseCount::updateAll()` |
| 8 | 新着通知 (コメント) |
| 9 | 新着通知 (メディア) |

## 案件のライフサイクル
- 「削除」、または「終了」。`queue=1`
- 1ヶ月後に「凍結」。`queue=2`
- 1日後に不要なログを削除する。
